# Images

These are named placeholder slots. To change an image in the app, just replace
the file here with your own — **keep the same filename** and no code changes are
needed. The files are referenced by name in components and in `content.ts` could
be extended to point elsewhere.

| File             | Used for                          | Referenced in                |
| ---------------- | --------------------------------- | ---------------------------- |
| `logo.png`       | Logo in the top navigation bar    | `app/components/Nav.tsx`     |
| `hero.jpg`       | Optional banner image slot        | available for a hero/landing |
| `background.jpg` | Optional page background slot     | available for backgrounds    |

The current files are tiny 1×1 transparent placeholders so nothing looks broken
before you drop in real artwork.
